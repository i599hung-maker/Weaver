# 解盤與報告產出 SOP（占驗派）

一張新命盤從進來到出報告的完整步驟。照著做，任何命盤都會得到同架構、同品質的報告。
規則依據見 `docs/specs/2026-07-08-zhanyan-paipan-design.md`。

---


## 命主提交模板（給命主資料時用這個）

```
幫我解一個新命主，照解盤 SOP 出完整版命書：
稱呼：
性別：男／女
國曆生日：YYYY-MM-DD
出生時間：HH:MM（24小時制，照鐘錶時間，晚子時引擎自動換日）
出生地：（預設台北）
【可選】文墨截圖：附了先定盤
【可選】命主背景：年齡／職業／現況
【可選】想問的問題：會在對應章節加重著墨並於應期直接回答
```

最簡一行也可：「解盤出完整版命書：1990-08-07 14:30 女 高雄」。

---

## 第〇步：定盤（有文墨截圖才做）

命主若提供文墨天機（S5VoG）截圖，先逐項核對引擎輸出：農曆與時辰（晚子時換日）、五行局、
命宮身宮、命主身主子斗、四化落宮、天馬（月支）、截空旬空、大限起訖順逆。
不一致就停下來查，一致才往下。

## 第一步：倒資料（一行指令）

```bash
npx tsx scripts/dump-chart.ts 1994-12-02 02:43 女 [經度]
```

得到：基本盤（真太陽時／農曆／五行局／命主身主子斗／四柱）、十二宮全星曜、
每個大限的疊宮（大官大財大妻災宮）與大限備註（幾層四化同星／交會）、
重點應期年份（權重≥2 或流命／災宮引動，含原因鏈）。
**報告裡的每一個數字和年份都只能來自這份輸出，不准手算、不准編。**

## 第二步：讀盤分析（怎麼解）

依序回答七個問題，答案就是報告的骨架：

1. **定格局**：命／財／官三方主星是什麼組合？
   （殺破狼＝開創變動型；紫府武相＝守成經營型；機月同梁＝幕僚企劃型；巨日＝口才是非型⋯）
   → 決定整份報告的基調：這個人該「開局」還是「經營」。
2. **找樞紐**：生年四化落在哪些宮？祿權科忌各管什麼？
   內宮（命福疾）vs 外宮（遷官財友）的分佈 → 一句話講出「好運在哪、坑在哪」。
   特別看：祿或忌坐命／坐福德、雙祿（化祿＋祿存）交會、祿忌同宮。
3. **拆命宮**：主星＋輔煞＋關鍵雜曜（截空、華蓋、天姚、空劫⋯）、命身同宮否、對宮是誰。
   → 性格主軸與內在矛盾。
4. **四主題三方四正**：本命（命宮）、財運（財帛）、事業（官祿）、愛情（夫妻），
   各取本宮＋對宮＋三合兩宮的星曜組合，寫出優勢 3-4 條、劣勢 3-4 條。
   來因宮落在哪個主題，就標記「此生主線」。
5. **挑關鍵大限**：看引擎的大限備註，優先講：現行大限、祿忌同星限、祿權相疊限、
   四化全疊限、大限入主題宮（如大限走夫妻＝感情十年）。挑 3 個。
6. **選重點應期年**：從引擎輸出篩選——
   - 已過的年份挑 2-3 個當「驗盤」（請命主對答案，建立信任）
   - 今年與未來 10 年內的全講
   - 更遠的只講權重 3 或流命引動的大年
   - 每個主題至少覆蓋一年；同年多重引動優先。
7. **推攻守與禁區**：進攻年＝流命引動吉宮＋祿權疊的年；收斂年＝災宮年、忌疊忌年、換限前一年。
   「別走的路線」從缺點組合推（每張盤不同）：
   例：貪狼＋破軍＋福德忌＝投機禁區；殺破狼＝原地不動禁區；空劫夾財＝無制度理財禁區；
   太陽忌在僕役＝替人作保／合夥不設防禁區。每條禁區必配「改走這條」。

## 第三步：產內容（怎麼寫）

報告章節與產出公式（完整版 full 模板，章序壹～伍）：

| 章節 | 產出公式 |
|---|---|
| 天賦類型命名 | 格局＋命宮主星特質 → 「形容詞＋名詞」稱號（例：多才多藝的破局開拓者／圓融聚財的掌局者）＋3-4 個格局標籤＋一句白話定調 |
| 關鍵詞雲 | 星曜→關鍵詞對照表（見下）聚合命宮三方四正，優缺各 10-13 詞，字級＝權重（廟旺主星最大） |
| 「你是⋯」4 段 | 從格局、財官結構、特殊雜曜（華蓋等）、命身關係各取一段，第二人稱身分句，每段末尾給一條建議 |
| 閃光點 3 ↔ 弱點 3＋練習 | 弱點要誠實點名星曜組合；每個弱點配一條「行為級」練習（48小時規則、分帳戶這種可執行的） |
| 命盤十二宮方圖 | 直接照引擎輸出排（含大限起訖、四化徽章、命身／來因標記） |
| 命格定調 | 第二步問題 1＋2 的答案，濃縮成一段 |
| 四主題＋優劣對照卡 | 第二步問題 4；desc 用白話講結構，優劣各 3-4 條吉／煞子彈點 |
| 大限走勢 3 卡 | 第二步問題 5；每卡：期間＋定性標題（蓄力期／上升期／高峰期⋯）＋為什麼 |
| 重點應期時間軸 | 第二步問題 6；每年格式固定：標題（白話一句）＋描述＋「為什麼」框（看到什麼→推論什麼→建議什麼）；過去年寫「回看驗證」 |
| 知命改命攻守守則 | 第二步問題 7；進攻年／收斂年兩欄年曆＋別走的路線卡（紅框）＋一句話總結 |

**語氣規範**：全程白話、第二人稱；缺點誠實直寫不美化；每個論斷都要能指回星曜／引動依據；
不用「劫難、註定、大凶」等宿命字眼——壞事講成「收斂年／禁區」，永遠給出口（知命改命）。

## 第四步：組頁面與輸出

```bash
cp reports/<既有完整版>.html reports/<slug>-full.html   # 沿用版型與 CSS（reports/ 不入庫，本機保存）
# 換 <body> 內容（hero 資料、盤面、各章文字），毛筆大標取 2 字命書名（呼應格局）
node scripts/export-pdf.mjs                                # 需要 PDF 時（記得把檔名加進腳本）
```

驗證：playwright 截圖檢查盤面與版面 → git commit → `open` 給命主看。

## 品質檢查清單（出貨前跑一遍）

- [ ] 盤面十二宮與 dump 輸出逐宮一致（星曜、亮度、四化、大限起訖）
- [ ] 應期年份、干支、虛歲全部來自引擎輸出
- [ ] 沒有任何盤上不存在的星曜或自創年份
- [ ] 每個弱點都有配練習；每條禁區都有「改走這條」
- [ ] 過去年份有 2-3 個可供命主驗盤
- [ ] 手機寬度檢查一次（雜曜在窄螢幕會隱藏，屬預期）

## 附錄：天賦類型命名要素

`（命宮主星特質形容詞）＋（格局角色名詞）`。角色名詞由格局定：
殺破狼→開拓者／破局者；紫府武相→掌局者／經營者；機月同梁→軍師／企劃家；
巨日→傳播者／評論家；日月→調和者。形容詞從命宮輔曜取（多才多藝、圓融聚財、深思熟慮⋯）。
